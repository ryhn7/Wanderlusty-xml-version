## Cara menggunakan

1. Clone repository ini
2. Buka project ini di IDE yang kalian gunakan (direkomendasikan Android Studio / Intelij IDEA)
3. Project langsung dapat di run di IDE yang kalian gunakan
4. Selamat mencoba!
